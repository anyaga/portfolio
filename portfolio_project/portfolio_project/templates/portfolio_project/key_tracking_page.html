Need of the client:

This project was brought to our --- by the Facilities Manager. The manager needed a better way to track keys given to differnt people in the department. Previously, the manager tracked all information about key placement through a spreadsheet. During graduation periods, it was difficult to keep track of keys that were and were not returned. This made it so several rooms needed to be re-keyed to maintain security.

The manager requested that we create a product in the Google ecosystem that allowed her to better track keys given, keys expired, and rooms that need to be rekeyed due to secruity concerns.

Project Description

I created a Google Appscript Project. This project starts by collect all key information we have by parsing spreadsheets and Google Form responses. For a key entry to be accepted, it needs to have the individuals full name, andrewID (school ID name), advisor(if a student), date key was given, date key is set to expire (graduation for students), the key number, and the room number for the key. These response are added to a spreadsheet in which each sheet is the expiration year. There is a main sheet that displays keys that are going to expire soon, expired keys, and improper key entry's.

A sidebar is displayed that allows you to work with the spreadsheet. This sidebar includes the following features:

  • Edit an entry
  • Delete an entry
  • Look up all keys checked out by one person
  • Email people that have expired keys or soon to expired keys
